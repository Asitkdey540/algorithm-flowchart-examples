BMI Classification System for Healthcare Assistant

Problem Statement

Classify people based on their Body Mass Index (BMI) into appropriate health zones.

BMI = (Height (m))² / Weight (kg)

BMI Categories:

  • Underweight: BMI < 18.5
  • Healthy Weight: 18.5 ≤ BMI ≤ 24.9
  • Overweight: 25 ≤ BMI ≤ 29.9
  • Obese: BMI ≥ 30

Inputs to the algorithm:

  • Height (in meters)
  • Weight (in kilograms)

Expected output:

  • BMI value
  • Health category classification

Algorithm:

  1. Step1: Start
  2. Step2: Read/Input Height, Weight
  3. Step3: BMI ← Weight / (Height * Height)
  4. Step4: Print BMI
  5. Step5: If BMI < 18.5 then Print "Underweight"
  6. Step6: If BMI >= 18.5 AND BMI <= 24.9 then Print "Healthy Weight"
  7. Step7: If BMI >= 25 AND BMI <= 29.9 then Print "Overweight"
  8. Step8: If BMI >= 30 then Print "Obese"
  9. Step9: End

Flowchart:

Control Structure Type: This algorithm uses Branching (Selection) control structure with multiple if-then conditions to classify BMI into different health categories.
