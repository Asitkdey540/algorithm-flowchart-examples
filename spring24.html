Nila's Coding Schedule System

Problem Statement

Determine Nila's coding duration based on rain duration and conditions:

  • If rain duration is even: code for 2 times the rain duration
  • If rain duration is odd: code for 3 times the rain duration
  • If rain duration is 0 or more than 12 hours: go to "sleep"

Input to the algorithm:

  • Rain duration (in hours)

Expected output:

  • Coding duration (in hours) OR "Sleep"

Sample Test Cases:

  • Input: 4 → Output: 8 (4 is even, so 4*2 = 8)
  • Input: 5 → Output: 15 (5 is odd, so 5*3 = 15)
  • Input: 0 → Output: Sleep (0 hours of rain)
  • Input: 18 → Output: Sleep (more than 12 hours)

Algorithm:

  1. Step1: Start
  2. Step2: Read/Input Rain_Duration
  3. Step3: If Rain_Duration = 0 OR Rain_Duration > 12 then Print "Sleep" and go to Step8
  4. Step4: If Rain_Duration MOD 2 = 0 then Coding_Duration ← Rain_Duration * 2
  5. Step5: If Rain_Duration MOD 2 ≠ 0 then Coding_Duration ← Rain_Duration * 3
  6. Step6: Print Coding_Duration
  7. Step7: End
  8. Step8: End

Flowchart:

Control Structure Type: This algorithm uses Branching (Selection) control structure with multiple if-then conditions to determine coding duration based on rain conditions.
